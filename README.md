Sistema de Informação referente ao Trabalho de Programação Orientada à Objetos.

O Trabalho terá como objetivo elaborar e codificar um sistema de controle de um Posto de Combustível.

Algumas funções:

- Controle Entrada e Saída de Combustível;
- Controle de Bombas de Combustível;
- Atualizar valores dos combustível;
- Teste de qual combustível compensa mais (Gasolina/Alcool);
- Relatório de Entrada e Saída conforme período.
